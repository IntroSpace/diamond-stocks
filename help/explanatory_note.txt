Пояснительная записка.

Свой модуль:
    для быстрой и удобной работы с API был реализован специальный модуль

Поля для ввода пользователя:
    данные, введённые пользователем, сохраняются в БД для удобства

Главное окно:
    вывод всей информации в удобном формате

Сайт компании:
    удобный просмотр официального сайта прямо в приложении

График изменения цен:
    для удобного сравнения цен

Экспорт данных:
    сохранение данных в Excel и CSV файлах

Выделение избранных компаний:
    чтобы не потерять и легко открыть любимую компанию

Заметки для избранных:
    можно оставить заметку для избранных компаний для удобства

Поиск компаний:
    можно найти нужную компанию по запросу

Рекоммендации:
    список рекоммендованных известных компаний
